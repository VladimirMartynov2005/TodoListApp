TodoList Console App

Консольное приложение для управления списком задач на C#.

Проект создан в рамках учебной практики для освоения основ C#, работы с Git и GitHub.

Функционал:

Описание:

1.Добавление новой задачи
2.Просмотр всех задач
3.Отметка задачи как выполненной
4.Удаление задачи по ID
5.Статистика по задачам (всего/выполнено/активно)
6.Показать только активные задачи
7.Показать только выполненные задачи
8.Выход из программы 

Запуск
Способ 1: Через Visual Studio (рекомендуется)

Требования
Visual Studio 2019/2022(https://visualstudio.microsoft.com/) с установленным компонентом ".NET desktop development"
.NET Core 3.1(https://dotnet.microsoft.com/download) или выше (устанавливается вместе с VS)

Инструкция
1. Клонируй репозиторий:
git clone https://github.com/VladimirMartynov2005.git
   
Дважды кликни по файлу TodoListApp.sln (если есть)
ИЛИ открой через Файл → Открыть → Проект/Решение и выбери TodoListApp.csproj
Запусти программу:
Нажми F5 для запуска с отладкой
ИЛИ Ctrl + F5 для запуска без отладки

Способ 2: Через терминал (командную строку)
Требования
.NET SDK (версия 3.1 или выше)
Установленный Git (для клонирования)

Инструкция
Клонируй репозиторий:
bash
git clone https://github.com/VladimirMartynov2005.git
Перейди в папку проекта:
bash
cd TodoListApp
Восстанови зависимости (если есть):
bash
dotnet restore

Собери проект:
bash
dotnet build

Запусти программу:
bash
dotnet run

Совет: Можно запустить без сборки отдельной командой:
bash
dotnet run --project TodoListApp.csproj

Альтернативный способ (запуск напрямую из папки bin)
После сборки можно запустить .exe файл:
bash
# Перейди в папку с собранным приложением
cd bin\Debug\netcoreapp3.1\  # или net8.0 и т.д.
# Запусти исполняемый файл
TodoListApp.exe
Важно: Путь к папке может отличаться в зависимости от версии .NET.
